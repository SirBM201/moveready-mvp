# Project MoveReady - Step 1A MVP Scope

Status: Locked draft  
Working name: Project MoveReady  
Final brand/domain: To be decided later

## Product Category

Project MoveReady is a global visa, travel, and relocation readiness platform. It is not only a travel app and not only a visa blog. The first product should help users understand realistic pathways, prepare correctly, and avoid costly relocation mistakes.

## Core Mission

Help users choose the right travel, visa, study, work, business, family, scholarship, or relocation pathway; prepare documents correctly; understand proof-of-funds and budget expectations; and generate trusted relocation readiness reports.

## Target Users

- Students seeking admission or scholarships
- Skilled workers comparing work visa routes
- Entrepreneurs and startup founders
- Digital nomads and remote workers
- Families planning relocation
- Visitors needing visa guidance
- People who need document readiness guidance
- People recovering from visa refusal
- Gulf-based migrants and emerging-market users planning their next move

## MVP Promise

The MVP should answer three practical questions:

1. Which country or route may fit my profile?
2. What documents, funds, budget, and insurance should I prepare?
3. What is my next-step relocation readiness plan?

## V1 Features

- Country and visa route checker
- User relocation profile questionnaire
- Country comparison
- Visa pathway explanation
- Document checklist generator
- Proof-of-funds readiness guidance
- Scholarship finder
- Travel and health insurance guide
- Relocation budget calculator
- Paid AI relocation readiness report
- Official-source-backed answer system
- Admin dashboard for managing routes, sources, and review tasks

## Not In V1

These are important but should not delay the first version:

- Live courier booking
- Flight, hotel, or taxi booking
- Full consultant marketplace
- Full document upload vault
- Automated embassy appointment booking
- Legal representation
- Full mobile app

## Phase 2 Features

- Visa refusal analyzer
- Notarization, apostille, and attestation guide
- Passport and document courier support
- Consultant marketplace

## Phase 3 Features

## MVP User Flow

1. User opens the app.
2. User selects a goal: study, work, business, visit, family, scholarship, or relocation.
3. User answers profile questions.
4. App suggests possible countries and route categories.
5. User selects a preferred route.
6. App shows overview, eligibility notes, document checklist, proof-of-funds guide, estimated budget, scholarship options, and insurance notes.
7. User generates a relocation readiness report.
8. User receives a next-step action plan and can save the route.

## Success Criteria

The MVP is successful if a user can:

- Enter a profile
- Compare possible routes
- Understand basic eligibility and risks
- Generate a useful checklist
- Estimate relocation budget
- Understand proof-of-funds expectations
- Find possible scholarships where relevant
- Generate a source-backed relocation readiness report
- See last verified dates and trust indicators

## Locked Direction

Project MoveReady MVP will be a source-backed global visa, relocation, scholarship, document, insurance, and budget readiness platform with paid AI-generated relocation readiness reports.
